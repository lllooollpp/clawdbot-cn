# 🦞 Clawdbot (中文适配版) — 深度适配国内生态的 AI 代理助理

如果你在寻找一个既能像 Claude 般强大，又能完美融入企业微信、飞书等国内办公生态，且拥有全中文支持的 AI 机器人框架，那么 **Clawdbot 中文适配版** 正是为你准备的。

🚀 **GitHub 仓库地址**：[https://github.com/lllooollpp/clawdbot-cn](https://github.com/lllooollpp/clawdbot-cn)

---

### 🌟 为什么选择这个项目？

**Clawdbot** 不仅仅是一个简单的对话机器人网关。它是一个**个人 AI 助手控制平面**，能够连接多端通讯软件，并赋予 AI 操作 Shell、浏览器、文件系统以及执行自动化任务的能力。

针对国内开发者，我进行了深度的**本地化重构与生态适配**：

#### 1. 📂 全面中文文档支持
原项目文档对国内访问不友好且只有英文。我专门搭建了**国内高速文档镜像**，并提供了完整的全中文使用指南：
*   **中文文档地址**：[http://101.35.228.254/docs/](http://101.35.228.254/docs/)
*   涵盖从零开始的向导配置、API 认证、以及各种插件的详细部署说明。

#### 2. 💬 深度国内办公生态适配
*   **企业微信 (WeCom) 插件**：
    *   原生支持消息回调 **AES 解密**与 **SHA1 签名验证**。
    *   支持文字、图片、媒体文件的双向传输，完美适配企业自建应用。
    *   内网穿透与反向代理深度优化。
*   **飞书 (Feishu/Lark) 插件**：内置 Feishu 渠道，支持国内主流团队协作工具。
*   **国产模型支持**：原生支持 **智谱 (ChatGLM 4.7)** 等国产大模型认证，满足合规与访问速度需求。

#### 3. 🛠️ 国内开发环境优化
*   **环境加速**：优化了针对国内网络环境的安装脚本与开发镜像。
*   **WSL2 深度适配**：在 Windows + WSL2 环境下通过 pnpm 即可实现丝滑的本地开发与部署。

---

### 🚀 核心黑科技特性

*   **多代理路由**：可以将不同的对话（如 Telegram 与企业微信）路由到完全隔离的 AI 代理。
*   **Agent 级工具链**：AI 可以帮你执行 Bash 指令、操作本地文件、进行网页抓取。
*   **语音唤醒与对话**：支持实时语音交互，将 AI 变成你桌面上的实体伴侣。
*   **实时 Canvas**：支持 AI 驱动的视觉工作区，实时渲染交互界面。

### 🛠️ 快速上手

推荐环境：**Node.js 22+**，**pnpm** 软件包管理器。

```bash
# 1. 克隆项目
git clone https://github.com/lllooollpp/clawdbot-cn.git
cd clawdbot-cn

# 2. 安装并构建
pnpm install
pnpm build

# 3. 启动交互式向导（中文引导）
pnpm clawdbot onboard
```

# TOOLS.md - 用户工具说明（可编辑）

此文件用于*你*对外部工具和约定的笔记。
它并不定义哪些工具存在；Clawdbot 内部已内置了工具。

## 示例

### imsg
- 发送 iMessage/SMS：描述收件人/内容，在发送前确认。
- 倾向于简短消息；避免发送机密信息。

### sag
- 文本转语音：指定语音、目标听众/房间，并说明是否需要流式传输。

添加你希望助手了解的关于你本地工具链的其他内容。

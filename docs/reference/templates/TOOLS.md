# TOOLS.md - 本地笔记

技能定义了*工具如何工作*。此文件用于*你的*特定信息——那些与你的设置独特相关的内容。

## 什么放在这里

例如：
- 相机名称和位置
- SSH 主机和别名
- 优选的 TTS 语音
- 音响/房间名称
- 设备昵称
- 任何环境相关的事项

## 示例

### 摄像头
- 客厅 → 主要区域，180° 广角
- 正门 → 入口，运动触发

### SSH
- 家庭服务器 → 192.168.1.100，用户：admin

### 文本转语音（TTS）
- 首选语音："Nova"（温暖，略带英音）
- 默认扬声器：厨房的 HomePod

## 为什么需要分开？

技能是共享的。你的设置属于你。将它们分开意味着你可以更新技能而不丢失你的笔记，共享技能而不泄露你的基础设施。
